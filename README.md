# GitCommitsParser

## What is it?
Java-приложение для парсинга git-лога.

## How it works?

#### Параметры запуска приложения:
1) --gitLogFile (-l) 
(путь до файла с git-логом)
2) --regex (-r)
3) --format (-f)
(json, html или plain. По умолчанию: plain)
4) --HTMLTemplatePath (-h)
(путь до HTML шаблона)
